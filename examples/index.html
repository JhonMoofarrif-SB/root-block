Uso Programático

Los toast se pueden crear por JavaScript usando el ToastManager:

// Métodos rápidos
showSuccess('¡Éxito!');
showError('Error!');
// Configuración avanzada
toastManager.show({
type: 'info',
title: 'Título',
position: 'top-center',
autoDismiss: 5000
});
